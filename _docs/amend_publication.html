Choosing a good publication ID

The application will allow you to enter any text as publication ID, but you should take care using consistent well-formatted IDs to ensure ITU OB database is in good shape.

  • Publication ID can include uppercase letters, numbers, and underscores.
  • If the publication is associated with an ITU recommendation, the ID should include recommendation code.
  • Here are some examples of well-formed publication IDs:
    • E212_MNC
    • X121_DCC
    • BUREAUFAX

This guide will show how to create an “amendment” message in a given ITU OB edition.

It is assumed that you know the title or ID of the publication you will amend, and you have opened Issue Editor for your desired OB edition.
