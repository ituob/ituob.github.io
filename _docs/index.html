Authoring at a glance

  1. Site authors use the ITU OB editor app to view the latest snapshot of ITU OB data and make changes to it.
  2. Changes to the data get integrated into ITU OB’s upstream data repository.
  3. When upstream data repository is updated, it builds and publishes a new version of ITU OB online.
  4. ITU OB online includes human-readable website, PDF deliverables, and datasets for programmatic access.
